# springboot-doma2-kotlin-example

The Example of SpringBoot Application.

## Composition

- Kotlin 1.6.0
- Java 17
- Maven(Wrapper) 3.6.3
- SpringBoot 2.6.1
- Doma2 2.50.0

## Upgrade Maven Wrapper

```sh
mvn -N io.takari:maven:0.7.7:wrapper
```
